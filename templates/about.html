About the Anomaly Detection Project

Project Goal

The goal of this project is to develop a robust anomaly detection system for video surveillance using deep learning models. This system aims to detect various abnormal activities such as theft, fighting, arson, and other criminal actions in real-time from surveillance videos. The anomaly detection model is trained on a diverse dataset and is capable of distinguishing between normal and anomalous behavior.

How I Achieved This

To accomplish this, I experimented with several state-of-the-art deep learning architectures for video classification. The models used include:

  • EfficientNetB7
  • MobileNetV2
  • InceptionNetV3
  • ConvLSTM2D with Batch Normalization and Dropout
  • GRU-based sequence models

Dataset Overview

The dataset used for this project is from the UCF-Crime Dataset, which consists of various videos categorized into 14 classes:

  • Abuse
  • Arson
  • Burglary
  • Fighting
  • Road Accidents
  • Shooting
  • Stealing
  • Arrest
  • Assault
  • Explosion
  • Normal
  • Robbery
  • Shoplifting
  • Vandalism
